Mohon dibaca:
1. Template tidak harus diawali dengan 'tmpl_'

2. Untuk membuat template, usahakan nama folder tidak ada white space atau spasi
contoh:
tmpl_yahoo - benar
facebook_ku - benar

tmpl yahoo - salah
facebook ku - salah

3. Untuk view template anda tinggal menambahkan ?tmpl=nama_folder_template
contoh:
- http://localhost/metu/?tmpl=tmpl_facebook
- http://localhost/metu/register/?tmpl=tmpl_twitter
