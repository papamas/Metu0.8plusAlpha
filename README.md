MeTu+
=======

MeTu+ atau dalam format yang panjang adalah “MediaTutorial Plus” merupakan sebuah project percobaan dari MediaTutorial. Project ini diberi akhiran tanda “+” dengan harapan nantinya dapat memberikan nilai plus untuk MediaTutorial karena kita lihat situs web tutorial ini belum sempurna kalo hanya sharing tutorial tanpa sebuah project.

MeTu+ nantinya merupakan sebuah Social Network Cms yang dilengkapi dengan fasilitas-fasilitas seperti pada Facebook dan Google+. Semua tutorial Php dan Codeigniter nanti akan kita praktekkan dalam Cms ini. Sebetulnya dulu kita pernah membuat sebuah Structurecore untuk project yang sama, tetapi kemudian kita men-discontinued project Structurecore dan menggantinya menjadi MeTu+.

Sesuai rencana, MeTu+ ini selain berupa versi web (cms) nantinya juga dilengkapi dengan “Desktop Messenger” dan “Mobile Messenger”. Kalau dalam cms kita memperagakan Php dan Codeigniter, dalam Desktop Messenger kita akan menggunakan Java, C++, Flash, Air, atau bahasa pemrograman berbasis desktop lain yang kita bisa. Sedangkan untuk Mobile Messenger nantinya kita akan memperagakan Java ME dan Android.

Sehingga nantinya MeTu+ ini akan ada dalam 3 versi yang saling mendukung, versi cms, versi desktop app, dan versi mobile app dan kesemuanya akan open source.
